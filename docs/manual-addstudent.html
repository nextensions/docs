›คู่มือเกี่ยวกับข้อมูลนักเรียน

คู่มือเกี่ยวกับเรื่องทั่วไป

  • การเข้าสู่ระบบ
  • คู่มือการตั้งค่าสำหรับผู้ดูแลระบบ

คู่มือเกี่ยวกับข้อมูลนักเรียน

  • การเพิ่มข้อมูลนักเรียนเข้าใหม่
  • การบ้าน
  • การจัดการคะแนนความประพฤติ
  • บันทึกข้อมูลสุขภาพ
  • แก้ไขสถานะนักเรียน

คู่มือเกี่ยวกับการลงเวลา

  • การยืม-คืนบัตรสำรอง
  • บันทึกการลงเวลาของนักเรียนแบบ Manual และ แบบกลุ่ม
  • บันทึกการลงเวลาของบุคลากรแบบ Manual
  • บันทึกการลาของนักเรียนแบบรายบุคคล และ การลาแบบรายกลุ่ม
  • บันทึกการลาของครู
  • สร้างกิจกรรมโรงเรียน

คู่มือเกี่ยวกับการแจ้งข่าวสาร

  • แจ้งข่าวสาร GPA
  • การแจ้งข่าวสารแบบกลุ่ม แบบเดี่ยว แบบทั้งโรงเรียน

คู่มือเกี่ยวกับวิชาเรียน

  • ตารางสอน
  • จัดตาราง(นักเรียน)
  • จัดตารางสอน(ครู)
  • การลาระหว่างวัน
  • การเช็คชื่อนักเรียนรายคาบรายวัน
  • การยกเลิกการสอน
  • คู่มือการตั้งค่าตารางสอนสำหรับ Admin

คู่มือการประเมินพฤติกรรม

  • สถานะการประเมิน SDQ
  • ประเมินด่วน SDQ
  • นักเรียนมีความเสี่ยง SDQ
  • สถานะการประเมิน EQ
  • การประเมินด่วน EQ

คู่มือการรับสมัครนักเรียนและโปรแกรมช่วยจัดตารางสอน

  • การจัดตารางสอน
  • การเปิดรับสมัครการสอบ Admission
  • การเปิดรับสมัครวิชาเลือก

คู่มือการใช้ห้องพยาบาล

  • การใช้งานระบบห้องพยาบาล
  • การใช้งานระบบห้องสมุด

คู่มือการสั่งบัตรใหม่ผ่าน store

  • การสั่งซื้อบัตรใหม่ทาง Store

คู่มือเกี่ยวกับระบบศูนย์อาหาร

  • คู่มือการใช้งานระบบศูนย์อาหาร

คู่มือเกี่ยวกับครื่องสแกนบัตรและ Hardware ต่าง ๆ

  • สถานะของไฟที่ตัวเครื่องสแกนบัตร
  • คู่มือการใช้งานเกี่ยวกับเครื่องศูนย์อาหาร
  • เครื่องเช็คชื่อรายวิชา SIC

คู่มือการดาวน์โหลดและใช้งาน Application Nextschool

  • การใช้งาน Application Nextschool
  • การใช้งาน Application Nextschool (สำหรับผู้ปกครอง)
  • การใช้งาน Application Nextschool (นักเรียน)

คู่มือการใช้งานเว็บ Student.nextschool

  • การใช้งานระบบ Student.nextschool สำหรับนักเรียน

การเพิ่มข้อมูลนักเรียนเข้าใหม่

1. การเพิ่มข้อมูลนักเรียนเข้าใหม่

  • เมนูนักเรียน -> ข้อมูลนักเรียน
  • กดเครื่องหมาย + เพื่อเพิ่มข้อมูลรายบุคคล

2. แก้ไขข้อมูลอัพเดทรูปนักเรียน หรือ เบอร์โทรผู้ปกครอง

  • เมนูนักเรียน -> ข้อมูลนักเรียน
  • ค้นหาชื่อนักเรียน หรือ รหัสนักเรียนที่ต้องการแก้ไข
  • แก้ไขข้อมูลตามต้องการ

3. แก้ไขเบอร์โทรนักเรียน

  • เมนูนักเรียน -> ข้อมูลนักเรียน
  • ค้นหาชื่อนักเรียน หรือ รหัสนักเรียนที่ต้องการแก้ไข
  • อัพเดทเบอร์ที่ต้องการ

4. แก้ไขรูปภาพนักเรียน

  • เมนูนักเรียน -> ข้อมูลนักเรียน
  • ค้นหาชื่อนักเรียน หรือ รหัสนักเรียนที่ต้องการแก้ไข
  • อัพเดทรูปที่ต้องการ

5. แก้ไขเบอร์ผู้ปกครอง

  • เมนูนักเรียน -> ข้อมูลนักเรียน
  • ค้นหาชื่อนักเรียน หรือ รหัสนักเรียนที่ต้องการแก้ไข
  • กรณีเพิ่มเบอร์โทรผู้ปกครอง
