›คู่มือเกี่ยวกับการแจ้งข่าวสาร

คู่มือเกี่ยวกับเรื่องทั่วไป

  • การเข้าสู่ระบบ
  • คู่มือการตั้งค่าสำหรับผู้ดูแลระบบ

คู่มือเกี่ยวกับข้อมูลนักเรียน

  • การเพิ่มข้อมูลนักเรียนเข้าใหม่
  • การบ้าน
  • การจัดการคะแนนความประพฤติ
  • บันทึกข้อมูลสุขภาพ
  • แก้ไขสถานะนักเรียน

คู่มือเกี่ยวกับการลงเวลา

  • การยืม-คืนบัตรสำรอง
  • บันทึกการลงเวลาของนักเรียนแบบ Manual และ แบบกลุ่ม
  • บันทึกการลงเวลาของบุคลากรแบบ Manual
  • บันทึกการลาของนักเรียนแบบรายบุคคล และ การลาแบบรายกลุ่ม
  • บันทึกการลาของครู
  • สร้างกิจกรรมโรงเรียน

คู่มือเกี่ยวกับการแจ้งข่าวสาร

  • แจ้งข่าวสาร GPA
  • การแจ้งข่าวสารแบบกลุ่ม แบบเดี่ยว แบบทั้งโรงเรียน

คู่มือเกี่ยวกับวิชาเรียน

  • ตารางสอน
  • จัดตาราง(นักเรียน)
  • จัดตารางสอน(ครู)
  • การลาระหว่างวัน
  • การเช็คชื่อนักเรียนรายคาบรายวัน
  • การยกเลิกการสอน
  • คู่มือการตั้งค่าตารางสอนสำหรับ Admin

คู่มือการประเมินพฤติกรรม

  • สถานะการประเมิน SDQ
  • ประเมินด่วน SDQ
  • นักเรียนมีความเสี่ยง SDQ
  • สถานะการประเมิน EQ
  • การประเมินด่วน EQ

คู่มือการรับสมัครนักเรียนและโปรแกรมช่วยจัดตารางสอน

  • การจัดตารางสอน
  • การเปิดรับสมัครการสอบ Admission
  • การเปิดรับสมัครวิชาเลือก

คู่มือการใช้ห้องพยาบาล

  • การใช้งานระบบห้องพยาบาล
  • การใช้งานระบบห้องสมุด

คู่มือการสั่งบัตรใหม่ผ่าน store

  • การสั่งซื้อบัตรใหม่ทาง Store

คู่มือเกี่ยวกับระบบศูนย์อาหาร

  • คู่มือการใช้งานระบบศูนย์อาหาร

คู่มือเกี่ยวกับครื่องสแกนบัตรและ Hardware ต่าง ๆ

  • สถานะของไฟที่ตัวเครื่องสแกนบัตร
  • คู่มือการใช้งานเกี่ยวกับเครื่องศูนย์อาหาร
  • เครื่องเช็คชื่อรายวิชา SIC

คู่มือการดาวน์โหลดและใช้งาน Application Nextschool

  • การใช้งาน Application Nextschool
  • การใช้งาน Application Nextschool (สำหรับผู้ปกครอง)
  • การใช้งาน Application Nextschool (นักเรียน)

คู่มือการใช้งานเว็บ Student.nextschool

  • การใช้งานระบบ Student.nextschool สำหรับนักเรียน

การแจ้งข่าวสารแบบกลุ่ม แบบเดี่ยว แบบทั้งโรงเรียน

เมนูแจ้งข่าวสาร

1. แบบเฉพาะบุคคล

  • กรอกชื่อนักเรียน (สามารถใส่ชื่อได้มากกว่า 1 คนขึ้นไป)
  • กรอกหัวข้อที่ต้องการเรียกพบ เช่น แจ้งนักเรียนให้มาที่ห้องกิจการนักเรียนในช่วงพักเที่ยงของวันนี้ เป็นต้น
  • พร้อมกับเนือหาที่ต้องการเรียกพบ เช่น นักเรียนได้ทุน นักกีฬาโรงเรียน หรือทำผิดกฏกับทางโรงเรียน เป็นต้น
  • เลือกประเภทที่ต้องการส่งข้อความ
  • กดปุ่ม ส่งข้อความ

2. แบบเฉพาะกลุ่มนักเรียน

  • เลือกระดับชั้นเรียน (สามารถเลือกระดับชั้นเรียนได้มากกว่า 1 ห้องขึ้นไป)
  • กรอกหัวข้อที่ต้องการเรียกพบ เช่น แจ้งย้ายห้องเรียน ทัศนศึกษา เป็นต้น
  • พร้อมกับเนือหาที่ต้องการเรียกพบ เช่น ย้ายห้องเรียนเดิมเรียนห้อง 1203 ย้ายมาgiupoที่ห้อง 2311 เป็นต้น
  • เลือกประเภทที่ต้องการส่งข้อความ
  • กดปุ่ม ส่งข้อความ

3. แบบเฉพาะบุคลากร

  • เลือกกรอกชื่อบุคลากร (สามารถเพิ่มชื่อบุคลากรได้มากกว่า 1 คนขึ้นไป)
  • กรอกหัวข้อที่ต้องการเรียกพบ เช่น ประชุมงานทัศนศึกษา ประชุมงานวิชาการ เป็นต้น
  • พร้อมกับเนือหา เช่น จะเลือกระดับชั้นที่จะให้ไปทัศนศึกษา เป็นต้น
  • เลือกประเภทที่ต้องการส่งข้อความ
  • กดปุ่ม ส่งข้อความ

4. แบบเฉพาะกลุ่มบุคลากร

  • เลือกกลุ่มงานบุคลากร (สามารถเลือกกลุ่มงานบุคลากรได้มากกว่า 1 กลุ่มขึ้นไป)
  • กรอกหัวข้อที่ต้องการเรียกพบ เช่น ทัศนศึกษาต่างประเทศ เป็นต้น
  • พร้อมกับเนือหา เช่น ต้องการให้มาทุกคนเพื่อต้องการทราบยอดผู้ที่จะไปทัศนศึกษาครั้งนี้ เป็นต้น
  • เลือกประเภทที่ต้องการส่งข้อความ
  • กดปุ่ม ส่งข้อความ
