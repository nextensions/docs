ครู/อาจารย์

  • การเข้าสู่ระบบ
  • การจัดตารางสอน
  • การเพิ่มข้อมูลนักเรียนเข้าใหม่
  • บันทึกการลงเวลาของนักเรียนแบบ Manual และ แบบกลุ่ม
  • บันทึกการลงเวลาของบุคลากรแบบ Manual
  • การยืม-คืนบัตรสำรอง
  • บันทึกการลาของนักเรียนแบบรายบุคคล และ การลาแบบรายกลุ่ม
  • แจ้งข่าวสาร GPA
  • การจัดการคะแนนความประพฤติ
  • การแจ้งข่าวสารแบบกลุ่ม แบบเดี่ยว แบบทั้งโรงเรียน
  • ตารางสอน
  • จัดตารางสอน(ครู)
  • จัดตาราง(นักเรียน)
  • การลาระหว่างวัน
  • การบ้าน
  • บันทึกการลาของครู
  • สร้างกิจกรรมโรงเรียน
  • บันทึกข้อมูลสุขภาพ
  • การเช็คชื่อนักเรียนรายคาบรายวัน
  • การยกเลิกการสอน
  • การประเมินด่วน EQ
  • สถานะการประเมิน EQ
  • การเปิดรับสมัครการสอบ Admission
  • การเปิดรับสมัครวิชาเลือก
  • การใช้งานระบบห้องพยาบาล
  • การใช้งานระบบห้องสมุด

การเช็คชื่อนักเรียนรายคาบรายวัน

  • เมนูวิชาเรียน -> Attendence Check
    • คลิ๊กที่ เลือก Section Code (หมายเหตุ:Section จะเลือกได้เฉพาะอาจารย์ที่มีคาบสอนในวันเท่านั้น เช่น วันจันทร์มีสอน 3 คาบ อาจารย์ที่สอนจะเห็นทั้ง 3 คาบ แต่ถ้าหากอาจารย์ไม่มีสอนในวันจันทร์ในตัวเลือก Section Code ก็จะไม่มีให้เลือก)
