›คู่มือเกี่ยวกับเรื่องทั่วไป

คู่มือเกี่ยวกับเรื่องทั่วไป

  • การเข้าสู่ระบบ
  • คู่มือการตั้งค่าสำหรับผู้ดูแลระบบ

คู่มือเกี่ยวกับข้อมูลนักเรียน

  • การเพิ่มข้อมูลนักเรียนเข้าใหม่
  • การบ้าน
  • การจัดการคะแนนความประพฤติ
  • บันทึกข้อมูลสุขภาพ
  • แก้ไขสถานะนักเรียน

คู่มือเกี่ยวกับการลงเวลา

  • การยืม-คืนบัตรสำรอง
  • บันทึกการลงเวลาของนักเรียนแบบ Manual และ แบบกลุ่ม
  • บันทึกการลงเวลาของบุคลากรแบบ Manual
  • บันทึกการลาของนักเรียนแบบรายบุคคล และ การลาแบบรายกลุ่ม
  • บันทึกการลาของครู
  • สร้างกิจกรรมโรงเรียน

คู่มือเกี่ยวกับการแจ้งข่าวสาร

  • แจ้งข่าวสาร GPA
  • การแจ้งข่าวสารแบบกลุ่ม แบบเดี่ยว แบบทั้งโรงเรียน

คู่มือเกี่ยวกับวิชาเรียน

  • ตารางสอน
  • จัดตาราง(นักเรียน)
  • จัดตารางสอน(ครู)
  • การลาระหว่างวัน
  • การเช็คชื่อนักเรียนรายคาบรายวัน
  • การยกเลิกการสอน
  • คู่มือการตั้งค่าตารางสอนสำหรับ Admin

คู่มือการประเมินพฤติกรรม

  • สถานะการประเมิน SDQ
  • ประเมินด่วน SDQ
  • นักเรียนมีความเสี่ยง SDQ
  • สถานะการประเมิน EQ
  • การประเมินด่วน EQ

คู่มือการรับสมัครนักเรียนและโปรแกรมช่วยจัดตารางสอน

  • การจัดตารางสอน
  • การเปิดรับสมัครการสอบ Admission
  • การเปิดรับสมัครวิชาเลือก

คู่มือการใช้ห้องพยาบาล

  • การใช้งานระบบห้องพยาบาล
  • การใช้งานระบบห้องสมุด

คู่มือการสั่งบัตรใหม่ผ่าน store

  • การสั่งซื้อบัตรใหม่ทาง Store

คู่มือเกี่ยวกับระบบศูนย์อาหาร

  • คู่มือการใช้งานระบบศูนย์อาหาร

คู่มือเกี่ยวกับครื่องสแกนบัตรและ Hardware ต่าง ๆ

  • สถานะของไฟที่ตัวเครื่องสแกนบัตร
  • คู่มือการใช้งานเกี่ยวกับเครื่องศูนย์อาหาร
  • เครื่องเช็คชื่อรายวิชา SIC

คู่มือการดาวน์โหลดและใช้งาน Application Nextschool

  • การใช้งาน Application Nextschool
  • การใช้งาน Application Nextschool (สำหรับผู้ปกครอง)
  • การใช้งาน Application Nextschool (นักเรียน)

คู่มือการใช้งานเว็บ Student.nextschool

  • การใช้งานระบบ Student.nextschool สำหรับนักเรียน

คู่มือการตั้งค่าสำหรับผู้ดูแลระบบ

ตั้งค่า Admin

  • เข้าไปที่เมนู ตั้งค่า
  • บุคลากร
    • กดปุ่ม สร้างผู้ใช้งาน
    • เลือกกลุ่มสิทธิ์ผู้ใช้งาน -> คำนำหน้า -> กรอกชื่อ นามสกุล -> เบอร์โทร
    • กดปุ่ม บันทึก
  • เข้าไปที่เมนู ตั้งค่า
  • ครูเวร
    • เพิ่มข้อมูลครูเวรประจำวัน -> วันที่ -> กดปุ่ม เพิ่ม
  • เข้าไปที่เมนู ตั้งค่า
  • กลุ่มผู้ใช้งาน
    • กดปุ่ม สร้างกลุ่มผู้ใช้งาน
    • ตั้งชื่อกลุ่ม -> กดปุ่ม บันทึก
    • กำหนดสิทธิ์ผู้ใช้งาน
      • กดที่รูป ดินสอเพื่อต้องการเปลี่ยนชื่อกลุ่ม
      • กดที่รูป ถูก เพื่อกำหนดสิทธิ์
      • เลือกกำหนดสิทธ์ให้แต่ละกลุ่ม (สามารถกำหนดต่างกันได้)
      • กดปุ่ม บันทึก
  • เข้าไปที่เมนู ตั้งค่า
  • ช่วงเวลาเรียน
    • กดปุ่ม เพิ่มช่วงเวลาเรียน
    • กรอกชื่อภาค -> เวลามาสาย -> เวลาไม่มา -> กดปุ่ม เพิ่ม
  • เข้าไปที่เมนู ตั้งค่า
  • วันหยุด
    • กดปุ่ม เพิ่มวันหยุด
    • กรอกชื่อวันหยุด -> รายละเอียด -> วันที่
    • กดปุ่ม เพิ่ม
  • เข้าไปที่เมนู ตั้งค่า
  • ปีการศึกษา/จัดชั้นเรียน
    • กดปุ่ม สร้างปีการศึกษา
    • เลือกปีการศึกษา -> วันเริ่มมาโรงเรียน -> วันสิ้นสุดการมาโรงเรียน
    • วันเริ่มการเรียนการสอน
      • สำหรับเช็คชื่อรายวิชา
      • วันสิ้นสุดการเรียนการสอน
      • กรอกชื่อภาคเรียน
      • กด Acive
      • กดปุ่ม สร้าง
    • เมื่อกดสร้าง ให้กำหนดจัดชั้นเรียนในภาคเรียน กดที่รูปปากกา
    • แก้ไขข้อมูลพื้นฐาน
    • จัดชั้นเรียน
    • ห้องเรียน และครูประจำชั้น
    • Grade Sync Service
  • เข้าไปที่เมนู ตั้งค่า
  • ความประพฤติ
    • หัวข้อความประพฤติ
      • กดปุ่ม ดาวน์โหลดไฟล์ต้นแบบ
      • กรอกตามแบบไฟล์ กรณีบันทึกไฟล์ บันทึกตามไฟล์ที่ดาวน์โหลด โดยไม่แก้ไขชื่อไฟล์
      • อัพโหลดไฟล์หัวข้อความประพฤติ (มาจากไฟล์ที่บันทึก)
      • กรณีต้องการเพิ่มเพียงหัวข้อเดียว สามารถทำได้เช่นกัน โดย
        • กดปุ่ม สร้างหัวข้อความประพฤติ
        • เพิ่มหัวข้อ -> ระบุประเภทความประพฤติ -> เพิ่ม/ลด คะแนน
        • สถานะ -> กดปุ่ม สร้าง
    • เงื่อนไขการตัดคะแนนอัตโนมัติ
      • กดปุ่ม สร้างเงื่อนไขการตัดคะแนน
      • กรอกช่วงเวลาเริ่มต้น-สิ้นสุด
      • เลือกหัวข้อความประพฤติ -> เพิ่ม/ลด คะแนน -> ระบุสถานะ
      • กดปุ่ม สร้าง
    • คะแนนความประพฤติเริ่มต้น
      • กรอกคะแนนความประพฤติเริ่มต้น
      • กดปุ่ม แก้ไข
  • ข้อมูลโรงเรียน
    • เลือกไฟล์ตราสัญลักษณ์ของโรงเรียน
    • ด้านขวาให้ปักหมุดที่ตั้งของโรงเรียน (เพื่อสะดวกในการค้นหาที่อยู่ของโรงเรียน)
  • กลุ่ม
    • กดปุ่ม เพิ่มกลุ่ม
    • กรอกชื่อกลุ่ม เช่น ทัศนศึกษา, แข่งกีฬา, ร.ด.
    • บุคคลในกลุ่ม (เพิ่มได้มากกว่า 1 คนขึ้นไป)
