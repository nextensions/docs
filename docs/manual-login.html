การเข้าสู่ระบบ

ระบบจะใช้งานบน Web Browser บนคอมพิวเตอร์หรือโทรศัพท์มือถือ แนะนำให้ใช้เป็น

Google Chrome

FrieFox

  1. เปิดโปรแกรม Google Chrome พิมพ์ URL ดังนี้ app.nextschool.io แล้วกด Enter เข้าไปยังเว็ปไซต์(หรือกดที่ลิ้งของเราได้เลย)

  1. เมื่อเข้ามาถึงหน้าเว็บไซต์ให้กดปุ่มสีฟ้า "ทดสอบเข้าสู้ระบบด้วยเบอร์มือถือ " เพื่อเข้าสู่ระบบ

  1. กรอกเบอร์มือถือ แล้วกดที่ "ตรวจสอบเบอร์มือถือ"

  2. กดที่ "ขอรหัสผ่าน OTP เข้ามือ"

  3. จะมีรหัสผ่านส่งมาที่ข้อความมือถือ กรอกรหัสผ่านที่ได้จำนวน

  4. ตัวแล้วกดที่ " VERIFY CODE"
